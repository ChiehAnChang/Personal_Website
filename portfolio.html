ML Travel Predictor

Tools Used: Python, pandas, scikit-learn

Description: This project focused on building a machine learning model capable of predicting travel destinations based on user preferences and historical travel data. The dataset included various features such as demographic data, travel history, and budget preferences, which were processed using pandas for feature engineering and cleaning. The model was implemented using scikit-learn, leveraging algorithms like Random Forest and Gradient Boosting for high accuracy.

Results: The model achieved an 85% accuracy rate, demonstrating robust predictive performance. It was tested on a dataset of over 10,000 user records and provided insightful recommendations that could potentially enhance customer experience in travel planning.

Community Hub

Tools Used: React, Node.js, Material-UI

Description: The Community Hub platform was designed to foster collaboration among students by enabling them to share their projects, ideas, and opportunities. The frontend was built with React and styled using Material-UI for a modern, responsive design. The backend, developed with Node.js, managed user authentication, post creation, and comment functionality. A MongoDB database stored the project's data efficiently.

Results: The platform saw an increase in student engagement, with over 500 active users within the first three months. Feedback highlighted its ease of use and impact on connecting students with similar interests and project goals.

UX Prototype

Tools Used: Figma, User-Centered Design (UCD)

Description: This project aimed to design a shared-ride service prototype that addressed common user pain points. Using UCD principles, the process began with user research to identify needs and expectations. The prototype was developed in Figma, showcasing features like real-time ride tracking, cost splitting, and an intuitive booking interface.

Results: The design received positive feedback during user testing sessions, where participants noted its clarity and ease of navigation. It was estimated that implementing this design could reduce development costs by 20% due to its detailed and actionable layout.
